## Overview

Elephas Advanced is a personal AI writing assistant that works across every macOS application, providing AI-powered writing help, text generation, summarization, and translation wherever you write without switching to a separate AI chat interface. It integrates with the system clipboard and text selection to make AI assistance available in any context — email clients, document editors, messaging apps, code editors, and any other application that handles text.

System-wide AI access activates through a keyboard shortcut or text selection menu in any application, sending selected text to AI models for rewriting, expanding, summarizing, translating, or responding without leaving the current application. The AI response replaces or supplements the selected text directly in place, keeping the workflow within the application where you are working.

Multi-model support connects to OpenAI GPT models, Anthropic Claude, Google Gemini, and local models running on your hardware, providing flexibility in choosing the AI model that best fits each task and your privacy preferences. Local model support ensures that sensitive text can be processed without sending data to external servers. Model switching is available per-request.

Custom personas define specialized AI behaviors for different writing contexts — a professional email editor, a creative writing coach, a code reviewer, a social media copywriter — that apply consistent tone, style, and approach to AI responses. Personas save the repetitive instruction-giving that generic AI interfaces require for recurring writing tasks.

For writers, professionals, and anyone who wants AI writing assistance available in every application without context switching to a separate chat interface, Elephas Advanced is the system-wide AI tool that brings language model capability to every text field on your system.

## Performance

Elephas balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
